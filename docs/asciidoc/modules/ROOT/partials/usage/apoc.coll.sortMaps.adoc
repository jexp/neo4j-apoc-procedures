The following sorts a list of maps in reverse alphabetical order by the key `name`:

[source,cypher]
----
RETURN apoc.coll.sortMaps([
    {name: "Lionel Messi"},
    {name: "Cristiano Ronaldo"},
    {name: "Wayne Rooney"}
], "name") AS output;
----

.Results
[opts="header",cols="1"]
|===
| Output
a|
[source,json]
----
[
    {
      "name": "Wayne Rooney"
    }
    ,
    {
      "name": "Lionel Messi"
    }
    ,
    {
      "name": "Cristiano Ronaldo"
    }
]
----
|===
