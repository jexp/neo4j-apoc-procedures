[[nlp]]
== Natural Language Processing (NLP)

[abstract]
--
This chapter describes procedures that can be used for added Natural Language Processing (NLP) functionality to graph applications.
--

The procedures described in this chapter act as wrappers around cloud based Natural Language APIs.
These procedures can extract entities, key phrases, and categories from text stored as node properties.
They can also compute the sentiment of a text document.
